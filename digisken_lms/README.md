# DigiSken LMS

A full-stack Learning Management System with Flutter frontend and Flask backend.

## Project Structure

```
digisken_lms/
├── flask_backend/     - REST API backend
└── flutter_app/       - Mobile application
```

## Getting Started

### Backend Setup

1. Navigate to flask_backend
2. Install dependencies: `pip install -r requirements.txt`
3. Run the server: `python run.py`

### Frontend Setup

1. Navigate to flutter_app
2. Get dependencies: `flutter pub get`
3. Run the app: `flutter run`

## Features

- Course listing and details
- Lesson viewing
- User-friendly interface
- Clean architecture pattern
